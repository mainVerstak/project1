Актуальность и устаревание объявлений

При наличии у объявления параметра expire-date для Yandex-стандарта или DateEnd для Avito-стандарта в XML-фиде, объявление будет снято с публикации по достижении этой даты.

При отсутствии вышеупомянутых параметров, объявление снимается с публикации в момент исчезновения упоминания о нем в XML-фиде.

Оба способа совместимы в одном XML-фиде для разных объявлений.
